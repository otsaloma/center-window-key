AGENTS.md
=========

Center Window Key is a GNOME Shell extension to add a keybinding to
center the currently focused window. The main implementation is in file
`extension.js`.

## General Instructions

* We're targeting GNOME 49 and later
* We're targeting Wayland only

## JavaScript

* Always use const where applicable
* Indent with four spaces
* Use double-quotes for strings
* Use private underscore-prefixed members
